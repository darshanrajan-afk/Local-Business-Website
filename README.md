# ☕ Matteo's Café — Local Business Website

A professional, animated single-page website for a neighbourhood café in Indiranagar, Bangalore.  
Built with plain HTML, CSS, and vanilla JavaScript — no frameworks, no dependencies.

---

## 🌐 How to Run

Just open `matteos-cafe-v2.html` in any browser. No setup needed.

---

## ✨ Key Features

- Dark coffee-themed UI with gold accents
- Loading screen, scroll animations & floating particles
- Tabbed menu (Coffee / Food / Desserts)
- Review carousel, gallery, and reservation form
- Sticky WhatsApp button + Google Maps link
- Fully mobile responsive
